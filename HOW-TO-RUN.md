# 🚗 RideBazzar - How to Run Your Project

## ⚡ Quick Start (2 Steps)

### Step 1: Start Backend Server

```powershell
# Open PowerShell/Terminal and run:
cd c:\srushti\rideBazzar\RideBazzar-Complete
cd backend
npm run dev
```

**You'll see this when it's working:**

```
🚗 RideBazzar Server running on port 5000
📡 API URL: http://localhost:5000/api
✅ Database connected successfully
```

### Step 2: Open Frontend

```powershell
# In another terminal (or double-click the file):
start home.html
```
