COMMON ISSUES & QUICK FIXES
================================================================================

"Build failed" error?
→ Check backend/package.json exists
→ Check backend/server.js exists

"Cannot connect to database"?
→ Wait 5 minutes (Railway takes time to provision)
→ Verify DB_HOST = mysql.railway.internal
→ Check DB_PASSWORD is correctly copied

"SMTP email not working"?
→ Need Gmail App Password (16 characters)
→ Create at: https://myaccount.google.com/apppasswords
→ Must have 2FA enabled first

"404 page not found"?
→ Make sure to add /home.html at end of URL
→ Example: https://ridebazzar-xxxx.railway.app/home.html
